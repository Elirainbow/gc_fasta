# GC_FASTA

Programa simple en Python para leer una secuencia de ADN desde un archivo FASTA y calcular su contenido de GC.

## Requisitos

Este proyecto usa `uv`.

## Archivos

- `main.py`: programa principal.
- `secuencia.fasta`: archivo de entrada con una secuencia de ADN.
- `pyproject.toml`: configuración del proyecto.

## Uso

```bash
uv run python main.py
```

## Ejemplo de salida

```text
La secuencia tiene 60.00% de GC
```
